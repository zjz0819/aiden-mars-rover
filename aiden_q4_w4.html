⚡ Try This!

Create a counter that increases by 1 each loop, prints the count, and resets to 0 when it reaches 10:

  1. Open Arduino IDE or upload to your board
  2. Copy the code from the right
  3. Upload to your Arduino board
  4. Watch the Serial Monitor (if using Serial output)
  5. Experiment by changing values or timing!
⚙️ Arduino Code
ROBO's Week 4 Sketch
int count = 0; void setup() { Serial.begin(9600); } void loop() { count++; Serial.print("Count: "); Serial.println(count); if (count >= 10) { count = 0; Serial.println("RESET!"); } delay(500); }

Type this Arduino code and watch ROBO come alive!

🌟 Hint 1: Arduino code is EXACT! Check spelling, parentheses, and semicolons. Every statement ends with ; and function calls use () with pins and values inside.
🌟 Hint 2: The Arduino IDE color-codes your words! Blue words are keywords, orange are functions, and numbers stand out. If something turns red, you have a syntax error — check parentheses and semicolons first!
🌟 Full answer: No shame — even NASA engineers need help! Copy the code exactly as shown, then upload to your board. If you get an error, the Arduino IDE will tell you which line is wrong. Fix it and try again!
