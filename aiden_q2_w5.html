🟠 MISSION 2 · BUILD
🔒 LOCKED
⏱ About 20 minutes

Build a basic line follower — if color sensor sees black, go straight; if it sees white, turn to find the line again.

⚡ Try This!

Build this in LEGO Education SPIKE App:

  1. Connect your SPIKE Prime robot to the SPIKE app
  2. Start a new project
  3. Drag the blocks you learned into the coding area
  4. Click the green play button to run your code
  5. Did SPIKE do what you expected? Try tweaking the numbers!
🎬 Block Sequence
Sample: Motor Forward
when started
forever
if color sensor = black then
Motor A + B run forward
else
Motor A turn right

Follow the black line on the mat!

🌟 Hint 1: Motor blocks are usually red/orange in the SPIKE app. Look for the "Motor" section in the block palette!
🌟 Hint 2: You need to specify the motor port (usually A, B, C, D, E, or F) and the action (run for rotations, run for seconds, run at speed, etc.).
🌟 Full answer: No shame — ROBO needed help learning too!
Use: Motor [port] run for [X] rotations to move, and wait [X] seconds to pause.
