While Loops — Patrol Mode
while loops run until a condition becomes false
WHILE BOOLEAN CONDITION
One New Concept
Loops until condition false
Mission Theme
Fuel-based patrol
Success Signal
Fuel countdown while loop

"The rover patrols Mars terrain until its fuel runs out. We don't know how many steps in advance — it depends on each drive. A for loop can't handle this. A while loop checks a condition BEFORE every run — and stops when false."

⏱ Session Timeline

Debrief W4
"For loops: fixed count. What if the rover should stop when fuel hits zero — but you don't know how many drives that takes?"
0:05–0:20

0:20–0:50
Build Patrol Simulator

1:05–1:10
Debrief
"Name a real-life situation where you'd use a while loop and one where you'd use a for loop."
🧠 Key Concepts
▸
while loop — runs while condition is True, stops when False
▸
Boolean — True or False — that's it
▸
Decrement — fuel -= 15 (same as fuel = fuel - 15)
▸
Infinite loop — while True: — runs forever until Ctrl+C
▸
break — immediately exits any loop
patrol_mode.py
# Rover Patrol System fuel = 100 distance = 0 while fuel > 0: fuel -= 15 distance += 50 print("Fuel:", fuel, "% | Distance:", distance, "m") if fuel <= 20: print("⚠️ LOW FUEL WARNING") print("Patrol complete. Total distance:", distance, "meters")

Clear path.") elif terrain == "icy": print("DANGER: Motor lock engaged.") elif terrain == "volcanic": print("CRITICAL: Evacuating sector!") else: print("Unknown terrain.
