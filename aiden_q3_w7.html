⚡ Try This!

Create a rover dictionary with 3-4 keys:

  1. Use keys like: name, battery, speed, location
  2. Print each value with a label
  3. Update the location and print again
  4. Loop through keys and values!
🐍 Python Code
ROBO's Status Dictionary
rover = {
  "name": "ROBO",
  "battery": 85,
  "speed": 5,
  "location": "Crater A"
}

print(rover["name"])
rover["location"] = "Crater B"
print(rover["location"])

Keys are labels, values are data. Use meaningful names!

🌟 Hint 1: Dictionary syntax: "key": value (notice the colon!). Keys are usually strings, values can be anything!
🌟 Hint 2: Access data with square brackets and the key name. rover["battery"] gets the battery value!
🌟 Full answer: Create a dict with at least 3 keys. Update one value. Print a few keys. You should see the updated data!
