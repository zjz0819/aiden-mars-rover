01 ROBO the Rover — Character & Expressions
Ready for today's mission, Aiden! 🚀
ROBO's Mood

ROBO is a named rover character who lives on every lesson page and reacts to what Aiden does. ROBO replaces the generic emoji theming with a consistent personality kids can follow week to week.

Each lesson opens with a short comic-style panel showing ROBO in trouble on Mars. Aiden's code is what saves him — making every lesson feel consequential, not like homework.

ROBO's expressions are pure SVG + CSS — zero external images, zero loading time. The 4 states map directly to lesson phases:

😐
NeutralPage load, between missions
🤩
ExcitedTry It phase — code time!
🤔
ThinkingLearn It phase — "process this..."
🎉
CelebratingAll checkboxes done — dances on screen!
💡 ROBO also has a unique weekly quote that continues the Mars story arc. By Week 40, Aiden will have followed ROBO's full journey across the planet.

📍 Mars Map Dashboard

The master dashboard replaces the current progress table with a top-down Mars terrain map. Each zone is a different visual region. Week nodes are landing pads — completed ones show Aiden's mission patch, the current one pulses as a beacon, future ones are dim/locked. The rover leaves tire tracks as it moves across the planet.
